Ⅰ- تعتبر عملية منجزة بالبلاد التونسية :

- فيما يخص البيع : كل عملية تم إنجازها وفق شروط تسليم البضاعة بالبلاد التونسية.

- فيما يخص أي عملية أخرى : كلما وقع إستعمال أو إستغلال الخدمات المسداة أو الحق المباع أو الشيء المستأجر بالبلاد التونسية. 

Ⅱ- تعتبر البضاعة المعدة للتصدير مسلمة بالبلاد التونسية بالنسبة لبانع ما إذا لم يودع هذا الأخير الإعلام بالتصدير بإسمه.

Ⅲ- تعتبر البضاعة الموردة مسلمة بالبلاد التونسية إذا وقع تسليمها لشخص آخر غير الشخص الذي استعمل رقم التعريفة الديوانية للقيام بالإجراءات الديوانية.
