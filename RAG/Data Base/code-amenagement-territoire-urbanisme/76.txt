باعث عقاري
شخص مادي أو معنوي ينج، قصد البيع أو الإيجار، بصفة اعتيادية أو من قبيل المهنة عمليات: - تقسيم و تهيئة أراضي مخصصة أساسا للسكنى - بناء و تجديد عقارات فردية أو نصف جماعية أو جماعية معدة للسكنى أو التجارة أو المهنة أو الإدارة
 يقوم بإنجاز تقسيم غير مصادق عليه أو بيع مقاسم منه يعاقب بالسجن لمدة تتراوح بين 16 يوما إلى 3 أشهر وبخطية تتراوح بين 500 دينار و20. 000 دينار أو بإحدى العقوبتين فقط. وفي صورة العود يصبح العقاب بالسجن متحتما.
