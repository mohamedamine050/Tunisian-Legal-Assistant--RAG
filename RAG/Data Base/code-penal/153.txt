يعاقب بالسجن مدة ثلاثة أعوام كل من يتعمد كسر أو رفع أو يحاول كسر أو رفع العلامات الخارجية كأشرطة أو طوابع أو معلقات أعدّتها سلطة إدارية أو عدلية لمنع الدخول لمح ّ لات أو رفع أشياء منقولة في صورة بحث عدلي أو جرد أو ائتمان أو عقلة. وإذا كان الحارس نفسه هو الذي كسر أو حاول كسر الأختام أو شارك في كسرها فالعقاب يكون بالسجن مدة خمسة أعوام وبخطية قدرها أربعمائة وثمانون دينارا.
