تعدّ معاطن أو مرابض أو أماكن مسيّجة كل أرض محوطة بحفير أو مواثيق أو بمشبك من القصب أو غيره أو بألواح أو بتخوم من نبات حي أو يابس أو بحائط كيفما كانت مواد تركيبه وكيفما كان ارتفاع وعمق وحالة قدم أو تهدم تلك الأنواع من المسيّجات ولو لم تكن لها أبواب تغلق بمفاتيح أو بغيرها أو كانت الأبواب ذات فرج أو مفتوحة عادة. والمعاطن أو المرابض غير القارة المعدّة لوضع الحيوانات بالأراضي بأي كيفية كان صنعها تعتبر أيضا مسيّجات.
