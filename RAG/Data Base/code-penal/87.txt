موظف عمومي
الشخص الذي تستخدمه الحكومة لأداء وظائف عامة.
 أو شبهه ويقبل بنفسه أو بواسطة غيره عطايا أو وعودا بالعطايا أو هدايا أو منافع كيفما كانت طبيعتها بدعوى الحصول على حقوق أو امتيازات لفائدة الغير ولو كانت حقا يعاقب بالسجن لمدة ثلاثة أعوام وبخطية قدرها ثلاثة آلاف دينار والمحاولة تستوجب العقاب. ويرفع العقاب إلى ضعفه إذا كان مرتكب الفعل موظفا عموميا أو شبهه.
