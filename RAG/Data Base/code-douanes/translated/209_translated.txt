The license is granted to persons residing in Tunisia, subject to the following conditions:

a) If the customs authorities can identify the imported and processed goods;

b) If the processing is sufficient such that the imported materials under this system cannot be returned to their original state at an economically acceptable cost;

c) When the purpose of using this system is not to circumvent the rules relating to origin and quantitative restrictions applicable to imported goods;

d) If this system is likely to provide the necessary conditions to contribute to the creation or maintenance of processing activities for goods within Tunisia without harming the essential interests of local producers of similar goods.
