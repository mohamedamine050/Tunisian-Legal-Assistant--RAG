مشارطة الإيجار لمدة معينة 
عقد
عقد
العقد هو اتفاق قانوني قائم يلزم الأطراف المتعاقدين ويحدد التزاماتهم وحقوقهم فيما يتعلق بالصفقة المعينة.
 يلتزم بمقتضاه المؤجر لقاء أجرة معينة أو قابلة للتعيين بأن يقوم بالسفرة أو السفرات التي يطلبها المستأجر بسفينة معينة إما خلال مدة متفق عليها أو خلال مدة غير محددة لكنها قابلة للتحديد.
