Articles 53, 54, and 55 of the Penal Code shall not apply to the penalties stipulated in Chapters VIII and IX of this Code, except for the cases referred to in Article 96 of this Code. Their application shall continue to apply to the penalties prescribed by the Penal Code to which this Code refers. However, if the offense is limited to a preparatory act for another offense, only the penalty related to the latter offense shall be applied.
