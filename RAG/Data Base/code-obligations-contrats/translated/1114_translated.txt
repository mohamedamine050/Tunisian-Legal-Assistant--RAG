The mandate does not entail remuneration unless otherwise stipulated. However, the mandate shall not be construed as gratuitous in the following instances:

Firstly: If the agent is appointed for a matter similar to that for which he was mandated.

Secondly: If the mandate is concluded between merchants in a commercial transaction.

Thirdly: If it is customary in the place to give remuneration for a matter similar to that for which the agent was mandated.
