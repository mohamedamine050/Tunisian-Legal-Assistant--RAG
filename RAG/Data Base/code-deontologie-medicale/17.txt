يحجر القيام بما يلي :ـ أولا : كل عمل من شأنه توفير نفع مادي غير مبرر أو غير جائز لمريض.ـ ثانيا : كل إرجاع نقدي أو عيني لمريض.ـ ثالثا : كل اقتسام للأتعاب بين الأطباء وخاصة كل ما يتعلق بإحالة أو بقبول أو بقسمة الأتعاب فيما بينهم بصورة خفية.ـ رابعا : كل سمسرة لأي شخص كان.ـ خامسا : قبول سمسرة لأجل أي عمل طبي وخاصة مقابل فحوص أو وصف أدوية وآلات أو إرسال إلى مركز نقاهة أو إلى مؤسسة صحية خاصة.
