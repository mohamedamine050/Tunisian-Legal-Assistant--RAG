في سبيل تطبيق هذا الفصل :

1) تعتبر سفينة : كل مركب عائمة مثل الكانسات والحاملات تباشر الملاحة البحرية والصهاريج وزوارق الجر مهما تكن بوسائلها الخاصة أو تجرها سفينة أخرى. 2) تعتبر سفينة صيد : كل سفينة لا تستخدم لأغراض إقتناص الموارد البحرية الحية. 3) تعتبر سفينة ترفيه : كل سفينة تستخدم للملاحة بدون غرض الكسب. 4) تعتبر سفينة شحن : كل سفينة ليست سفينة ركاب أو صيد أو ترفيه. 5) تعتبر سفينة ركاب : كل سفينة تقل ما يزيد عن اثني عشر راكبا. 6) تعتبر سفرا دوليا : كل رحلة تقوم بها سفينة تونسية ترسي بميناء أجنبي.
